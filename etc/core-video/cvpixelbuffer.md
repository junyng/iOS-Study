# CVPixelBuffer

### Overview

Core Video 픽셀 버퍼는 메인 메모리에 픽셀을 고정하는 이미지 버퍼다. 프레임을 생성하거나, 비디오를 압축 또는 압축 해제하거나, Core Image를 사용하는 애플리케이션은 모두 Core Video 픽셀 버퍼를 사용할 수 있다.
