# CMClock

### Overview

`CMSync` API는 애플리케이션과 장치를 동기화하는 데 사용할 수 있는 참조 클럭을 제공한다. 이 API는 또한 `CMClock`과 타이머 서비스와 관련된 함수 사이의 상대적인 드리프트를 감시하는 기능을 제공한다.
