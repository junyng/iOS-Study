# CMAttachment

### Overview

이 문서에 Core Media 첨부 프로토콜에 대해 설명한다. CMAttachmentBaerer는 키/값/모드 첨부 API의 객체를 지원하는 CF 기반 객체다. CF 객체를 CMAttachmentBearer 객체에 부착하여 추가 정보를 저장할 수 있다.
